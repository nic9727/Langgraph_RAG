# 学校规章制度

## 第一章 学分管理制度

### 1.1 学分要求
- 本科生毕业总学分要求：**160学分**
- 其中必修课学分不低于 **110学分**
- 选修课学分不低于 **30学分**
- 实践环节学分不低于 **20学分**

### 1.2 学分计算规则
- 理论课：每16学时计1学分
- 实验课：每32学时计1学分
- 课程设计：每周计1学分
- 毕业设计（论文）：计8学分

### 1.3 学分绩点
| 成绩区间 | 绩点 |
|---------|------|
| 90-100  | 4.0  |
| 85-89   | 3.7  |
| 82-84   | 3.3  |
| 78-81   | 3.0  |
| 75-77   | 2.7  |
| 72-74   | 2.3  |
| 68-71   | 2.0  |
| 64-67   | 1.5  |
| 60-63   | 1.0  |
| 60以下   | 0    |

## 第二章 考试与成绩管理

### 2.1 成绩构成
- 平时成绩占比：**30%**（包含出勤、作业、课堂表现）
- 期中考试占比：**20%**
- 期末考试占比：**50%**

### 2.2 补考与重修
- 课程成绩不及格的学生可参加一次补考
- 补考仍不及格者须进行重修
- 重修课程成绩按实际成绩记载，但标注"重修"
- 每学期重修课程不得超过3门

### 2.3 学业警告
- 一学期内不及格学分达到12学分者，给予学业警告
- 累计不及格学分达到25学分者，给予退学警告
- 累计不及格学分达到30学分者，予以退学处理

## 第三章 实习管理制度

### 3.1 实习要求
- 所有本科生须完成至少 **8周** 的专业实习
- 实习可在学校合作企业或自主联系单位进行
- 实习结束须提交实习报告和单位鉴定表

### 3.2 学校合作实习企业
- **华为技术有限公司** - 软件开发、人工智能方向
- **阿里巴巴集团** - 数据分析、云计算方向
- **腾讯科技** - 产品开发、游戏设计方向
- **百度公司** - 自然语言处理、搜索技术方向
- **字节跳动** - 推荐系统、短视频技术方向
- **科大讯飞** - 语音识别、智能教育方向
- **中国移动** - 通信技术、网络运维方向
- **中软国际** - 软件外包、项目管理方向

### 3.3 实习成绩评定
- 实习报告质量：**40%**
- 单位评价：**30%**
- 答辩表现：**30%**

## 第四章 奖学金制度

### 4.1 奖学金类型
- **一等奖学金**：绩点排名前5%，奖金5000元
- **二等奖学金**：绩点排名前15%，奖金3000元
- **三等奖学金**：绩点排名前30%，奖金1500元
- **国家奖学金**：绩点排名前2%且无不及格，奖金8000元

### 4.2 评选条件
- 无违纪处分记录
- 所有课程成绩合格
- 积极参加课外活动和社会实践
